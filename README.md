# 夸夸墙 - 员工表扬系统

## 系统简介

"夸夸墙"是一个HR模块网站，用于员工之间的互相表扬和认可，营造积极向上的企业文化。

## 功能特点

### 1. 权限管理
- **高级用户**：可访问所有页面，管理用户、导出报表、删除内容
- **普通用户**：只能查看展示页面
- **默认账号**：
  - ID: 101, 权限: 高级用户 (拥有所有权限)
  - ID: 102, 权限: 普通用户
  - ID: 103, 权限: 高级用户

### 2. 页面结构
- **第1页 - 登录页面**：柔和的淡紫色背景，支持中英文切换
- **第2页 - 管理页面**：导入Excel、导出3种报表、删除内容
- **第3页 - 展示页面**：挂板形式展示所有夸夸记录，支持上传图片
- **第4页 - 辅助页面**：预留扩展

### 3. 报表导出
- **报表1**：用户ID和权限列表
- **报表2**：当月所有录入记录
- **报表3**：用户ID和受表扬人ID的频次统计

## 使用说明

### 登录
1. 访问网站首页
2. 输入用户ID登录
3. 选择语言（中文/英文）

### 管理页面（高级用户）
1. **导入用户**：上传Excel文件（格式：第一列用户ID，第二列权限名称）
2. **导出报表**：点击对应按钮下载Excel
3. **查看展示页**：在右侧预览展示内容
4. **删除内容**：单条删除或批量删除

### 展示页面
1. 点击右下角 **+** 按钮添加夸夸
2. 输入内容（支持中文自动翻译英文）
3. 选择受表扬人
4. 可选上传图片
5. 查看历史记录（分页显示，每页10条）

## Excel导入格式

```
用户ID | 权限名称
101    | 高级用户
102    | 普通用户
103    | 高级用户
```
